# XGBoost model

##installation
* for XGBoost tutorial installation, it is available at : https://xgboost.readthedocs.io/en/latest/build.html

## How to run
* cd $ROOT/src/
* run `python xgboost_alt.py`
* result will be stored in $ROOT/data/rossmann_no_correction_scaled_another_Feat.csv

## Dataset
* The preprocessing for the dataset is done on the xgboost_alt.py, so no need to run an additional python file
