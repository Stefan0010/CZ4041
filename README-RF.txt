# RANDOM FOREST REGRESSOR

## How to run
* cd $ROOT/src/
* run run_rf_log.py to generate result by converting Sales to its logarithmic value
	* result will be stored in $ROOT/src/result1.csv and $ROOT/src/result2.csv
* run run_rf_std.lpy to generate result by normalizing Sales using its standard deviation
	*result will be stored in $ROOT/src/result1_std.csv and $ROOT/src/result2_std.csv
